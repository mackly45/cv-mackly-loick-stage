Projets & Réalisations

Projet de Distinction des Races de Chiens par Image

Ce projet utilise l'intelligence artificielle et le framework Blazor Server de Microsoft pour créer une application capable d'identifier les races de chiens à partir d'images. J'ai travaillé sur :

  • La collecte et le nettoyage des données
  • La conception du modèle de reconnaissance d'images
  • L'intégration de l'interface utilisateur avec Blazor Server

Projet sur la Notoriété d'une Page sur les Réseaux Sociaux

Dans ce projet, j'ai analysé la visibilité de plusieurs pages sur les réseaux sociaux en utilisant :

  • Des techniques de machine learning et d'analyse de données
  • L'analyse des tendances et des interactions

J'ai proposé des stratégies de publication pour améliorer la notoriété et l'engagement des utilisateurs.

Développement d'une Application de Gestion des Tâches en Python

Ce projet consistait à développer une application permettant de gérer les tâches quotidiennes des utilisateurs. L'application est construite en Python et utilise une base de données SQLite pour stocker les informations des tâches. Elle inclut des fonctionnalités de :

  • Création
  • Modification
  • Suppression
  • Consultation des tâches
