Projet de Distinction des Races de Chiens par Image

Ce projet utilise l'intelligence artificielle et le framework Blazor Server de Microsoft pour créer une application capable d'identifier les races de chiens à partir d'images. J'ai travaillé sur :

  • La collecte et le nettoyage des données
  • La conception du modèle de reconnaissance d'images
  • L'intégration de l'interface utilisateur avec Blazor Server
